Input Handlers

Change Happens "On Change"

For inputs, the change event doesn't fire until the input loses focus.

Enter something into the field, and then click somewhere else to trigger the event

Change Happens "On Focus"

Change Happens "On Blur"

The blur event fires when an element has lost focus.

Change Happens "On Key Up"

The keyup event fires when the user releases a key.

Select Handlers

Change Happens "On Change"

For select, the change event fires when the user selects a dropdown value.
